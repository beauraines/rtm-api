request/sign

RTM API Signature

This module provides the sign() function which can be used to sign the RTM API requests for a given set of parameters and RTMClient.

The parameters should be given as a single object with a set of key/value pairs as the object's properties. For example, the parameters: yxz=foo feg=bar abc=baz should be given as on object as:

{
 yxz: "foo",
 feg: "bar",
 abc: "baz"
}
